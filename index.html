HTML Cheatsheet

Tables

Attribute/Element Purpose
<table></table> to represent a two-dimensional table made of rows and columns
<tr></tr> used to add rows to a table before adding table data and table headings
<td></td> can be nested inside a table row element, <tr>, to add a cell of data to a table
<thead></thead> defines the headings of table columns encapsulated in table rows
<tbody></tbody> semantic element that will contain all table data other than table heading and table footer content. If used, <tbody> will contain all table row <tr> elements, and indicates that <tr> elements make up the body of the table. <table> cannot have both <tbody> and <tr> as direct children
<th></th> used to add titles to rows and columns of a table and must be enclosed in a table row element, <tr>
<tfoot></tfoot> uses table rows to give footer content or to summarize content at the end of a table
colspan indicates how many columns that particular cell should span within the table; default 1, will take positive integer up to 1000
rowspan indicates how many rows that particular cell should span within the table; default 1, will take positive integer up to 1000
